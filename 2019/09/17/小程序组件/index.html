小程序组件

1.cover-image

覆盖在原生组件之上的图片视图，可覆盖的原生组件同cover-view，支持嵌套在cover-view里

属性 类型 默认值 必填 说明 最低版本
src string 否 图标路径，支持临时路径、网络地址（1.6.0起支持）、云文件ID（2.2.3起支持）。暂不支持base64格式。 1.4.0
bindload eventhandle 否 图片加载成功时触发 2.1.0
binderror eventhandle 否 图片加载失败时触发 [2.1.0]

2.cover-view

覆盖在原生组件之上的文本视图

可覆盖的原生组件包括map、video、canvas、camera、live-player、live-pusher

3.movable-area

movable-view的可移动区域

4.movable-view

可移动的视图容器，在页面中可以拖拽滑动。movable-view必须在movable-area组件中，并且必须是直接子节点否则不能移动

属性

  • direction：定义movable-view的移动方向，属性值有all、vertical、horizontal、none
    • all：任意移动
    • vertical：仅y轴移动
    • horizontal：仅x轴移动
    • none：不能移动

5.bloc组件

block组件可以充当一个容器，其标签内容在页面中不会被渲染

6.picker组件

从底部弹起的一个选择器

通过设置mode属性指定其为什么模式的选择器，默认值为selector

当选择确定时触发bindchange事件，通过调用回调函数来处理，range设置选择数组。value设置数组索引值。

bindcancel取消选择时触发

index默认为0，通过改变当前index的值，使其渲染不同的选择。value值保存在e.detail中

7.picker-view组件

嵌入页面的滚动选择器，其中只可放置picker-view-cplumn组件，其他节点不会显示

indicator-style：设置选择器中间选中框的样式

8.picker-view-column

滚动选择器子项，仅可放置于picker-view中，其孩子节点的高度会自动设置成picker-view的选中框的高度一致
