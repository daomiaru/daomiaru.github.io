4.login

微信小程序提供了api让我么来处理登录请求，步骤为

1.调用wx.login()获取登录凭证code

2.调用wx.request()将code传到服务器

3.服务器调用微信登录校验接口(appid+appsecret+code)，获得session_key+openid

4.服务端生成3rd_session(相当于userid)，使用3rd_session为key，session_key+openid为value的形式存在服务器，并将3rd_session传送给客户端。

5.客户端本地存储3rd_session，登录成功！

6.下次登录时可以用3rd_session传输到服务器验证登录态是否过期，未过期则登录成功

这个需要跟服务端配合使用具体不多叙述。

5.动画

实现方式

  • 创建一个动画实例animation。
  • 调用实例的方法来描述动画
  • 最后通过动画实例的wxport方法导出动画数据传递给组件的animation属性

前两步是定义一个动画并设置要干什么，然后把这个设置好的规则导出扔给界面上的某个组件，让它按照这个规则来执行。

当然也可以给多个组件同时绑定相同的动画
