Ajax

1.什么是Ajax

ajax = Asynchronous JavaScript and XML(异步的javascript和XML)

ajax可以通过在后台与服务其进行少量的数据交换，使网页实现异步更新。这意味着可以在不重新加载整个网页的情况下，对网页的某部分进行更新

传统的网页（不使用AJAX）如果需要更新内容，必须重新加载整个页面

    同步和异步的区别
    同步：

    提交请求->等待服务器响应->响应完毕期间客户端浏览器不能干任何事->响应完毕

    异步：

    请求通过事件触发->服务器响应->响应完毕期间客户端浏览器可以做其他事->响应完毕

3.解决低版本浏览器缓存问题

浏览器的缓存问题是当浏览器读取一个文件时是按照url地址来解析的，当我们的文件内容发生变化时，浏览器读取的url地址也应该发生变化。我们这里有”url+?+任意字符”的方式来设置浏览器读取路径，而读取内容还是根据原来的url来决定的并且url地址也发生了变化。就能解决浏览器缓存问题

5.jQuery中的ajax

load()方法

  • 参数1：url

  • 参数2：data(可选)，如果有则为post方式提交数据，没有则为get

  • 参数3：callback(可选)

    回调函数中参数含义

    1.responseTxt - 包含调试成功时的结果内容

    2.statusTxt - 包含调试的状态可能是”success”、”notmodified”、”error”、”timeout”、”abort”或”parsererror”中的一个

    3.xhr - 经过jquery封装的XMLHttpRequest对象（保留其本身的所有属性和方法）

load()方法作用事从服务器加载数据，是一个简单但强大的ajax方法。.load()从服务器加载数据，然后将其放到html页面中
