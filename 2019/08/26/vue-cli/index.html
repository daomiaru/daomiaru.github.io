1.Vue CLI

什么是vue cli？

1.cli是command-line-interface，翻译为命令行界面，但是俗称脚手架

2.vue-cli是一个官方发布的项目脚手架

3.使用 vue-cli可以快速搭建vue开发环境的webpack配置

vue-cli2初始化目录结构

runtime+compiler和runtimeonly的区别

1.runtime+compiler: main.js

runtime+compiler模版解析过程

template->ast(抽象语法树)->render->vdom(虚拟dom)->ui

2.runtimeonly: main.js

runtimeonly模版解析过程

render->vdom->ui

runtimeonly代码量更小，性能更高

3.Vue-cli3

vue-cli3初始化项目

vue-cli3与2版本的区别

  • vue-cli3是基于webpack4打造，vue-cli2还是webpack3
  • vue-cli3的设计原则是“0配置”，移除的配置文件根目录下的，build和config目录
  • vue-cli3提供了vue ui命令，提供了可视化配置，更加人性化
  • 移除了static文件夹，新增了public文件夹，并且index.html移动到public中

vue-cli3初始化目录结构

若想修改默认配置，有三种方式

1.使用vue ui打开可视化工具进行修改

2.在根目录创建一个vue.config.js文件。以module.export导出方式在里面进行修改

3.直接修改配置文件，默认放在mode_module/@vue/cli-service/webpack.config.js下
